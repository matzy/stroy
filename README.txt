Build Requirements
stroy is a java program. You need java SDK 1.5 or newer.
All used libraries are in the source package.
There are project file for IntelliJ Idea and a ant script.
The libraries necessary to develop are in external/build external/production and
external/test directories.
Working with eclipse, netbeans or emacs should be no problem.
