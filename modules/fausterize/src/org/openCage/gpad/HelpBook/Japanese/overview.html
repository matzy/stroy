Overview

What is Fausterize?

Fausterize is a simple encrypting text editor.
When you open it the first time it shows a simple help message. To start using it select a file as encryption key: press decode. The next time you start fausterize it shows a mad poem made from lines of the novel faust be Goethe. Press decode to select your key file again. After decoding you can edit your text. The text is always saved encrypted in the background.
