Preferences

External Programs

stroy uses external programs to open and diff files or show as text. These programs can be changed.

Filetype Actions

Per filetype stroy lets you choose open, diff programs and the algorithm used to compare files.

Default Values for Directories

stroy remembers your last choice.

Filter

Filters keeps scm files/dir (.svn) and other noise files out of the diff. There is a preconfigured selection for the common cases. Add more as appropriate. A right click on a file delivers a specialized dialog to filter out that file or all files of that type.

Logs

stroy is not very talkative, it trys to do all operations as seamless as possible. But if you want to see what it does change the loglevel settings.
