Preferences

External Programs

stroy uses external programs to open and diff files or show as text. These programs can be changed.

Filetype Actions

Per filetype stroy lets you choose open and diff programs and the algorithm used to compare files.

Default Values for Directories

stroy remembers your last choice.

Filter

Filters keep scm files/dir (.svn) and other noise files out of the diff. There is a preconfigured selection for the common cases. Add more as appropriate. A right click on a file delivers a specialized dialog to filter out that file or all files of that type.

Logs

stroy is not very talkative. It trys to do all operations as seamless as possible. But if you want to see what it does change the loglevel settings.

Language

Choose the language of stroy. The default is just your OS setting, or pick English, German, Japanese or Spanish. Note: Spanish localization is not complete and falls back to English.

Update

stroy can check for updates. The interval is configurable.
